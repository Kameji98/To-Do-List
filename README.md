# To-Do List (Vanilla JS)

A simple to-do list app with localStorage persistence.

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Run Locally
Open `src/index.html` in your browser.

## Project Structure
- `src/index.html` — UI
- `src/styles.css` — styling
- `src/app.js` — logic

## What I practiced
- DOM events and dynamic rendering
- localStorage persistence
- Clean, minimal UI without frameworks
